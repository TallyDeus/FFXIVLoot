Raidplan Code or URL

Paste a raidplan code or full URL. Each step will be converted to a separate board.

Strategy boards are limited, so some Raidplan features will be lost or placed incorrectly in the conversion.

Make sure to double-check the generated boards in-game for accuracy.

  1. Objects outside the strategy board's aspect ratio will be cropped.
  2. All text will be center-aligned and each line of text will be a separate object.
  3. Text cannot be resized and is limited to 30 characters and 8 objects per board.
  4. The only objects that can have non-default colors are lines and line AOEs.
  5. Images, emoji, hand-drawn lines, and other status effects are not supported.
  6. Objects may be placed incorrectly or render in an incorrect order.
  7. Generated codes may not be able to be rendered in-game.
